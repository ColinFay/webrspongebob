## webrspongebob

Example repo of an app built with `webrcli` & `spidyr`.

Workflow to get there:

```bash
npm install -g webrcli
cd /tmp
webrcli init webrspongebob
cd ./webrspongebob
webrcli install spongebob
node index.js
```

If you git cloned this package:

```bash
git clone
cd webrspongebob
npm install
node index.js
```
